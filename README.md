## SpotiSaber

An Open-Source Spotify-to-Beat Saber Converter.

## Installation

Clone the repository

```bash
    git clone https://github.com/jackmerrill/SpotiSaber.git
```

Install modules

```bash
    yarn
    # or
    npm install
```

Fill the environment variables

```bash
    export SPOTIFY_ID=
    export SPOTIFY_SECRET=
    export NEXTAUTH_SECRET=
```

Run the dev server

```bash
    yarn dev
    # or
    npm run dev
```
